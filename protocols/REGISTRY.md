# A'Space Protocol Registry

This registry tracks the status and ownership of official communication protocols between agents.

| Contract | Status | Description | Owner |
|----------|--------|-------------|-------|
| `decision` | Stable | Human-in-the-loop approval requests. | Jerry |
| `intent` | Stable | Strategic goals and priorities from A0. | Beth |
| `order` | Stable | Specific tasks for execution. | Morty |
| `pulse` | Stable | Business metrics and health signals. | Summer |
| `uplink` | Stable | Weekly summary and sync signal. | Jerry |

## Versioning Rules
- **Schema Version**: Current global version is `1.0`.
- **Breaking Changes**: Requires a bump to `2.0` and a migration plan.
- **Additions**: Non-breaking additions can stay in `1.x`.
