# NON-GOALS : A'SPACE OS INTERFACE

Ce document liste explicitement ce que l'interface **ne doit PAS devenir**.

## Jamais Un Réseau Social
* Pas de "likes" ou "reactions"
* Pas de feed infini
* Pas de partage public
* Pas de comparaison avec d'autres utilisateurs

## Jamais Une App de Productivité Générique
* Pas de to-do lists "classiques" (on a déjà PARA)
* Pas de calendrier (Google Calendar existe)
* Pas de notes (Notion existe)

**Raison :** L'interface est un **cockpit**, pas un Swiss Army Knife.

## Jamais Une Plateforme de Monétisation
* Pas de freemium model
* Pas de version "Pro" cachée
* Pas de ads
* Pas de data selling

**Raison :** C'est un **outil souverain**, pas un produit SaaS.

## Jamais Un Système Fermé
* Pas de vendor lock-in
* Pas de format propriétaire
* Pas de dépendance à un seul cloud

**Raison :** Philosophie Solarpunk = **Open Source + Self-Hosted**.
